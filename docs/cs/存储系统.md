## 存储器概述

## 主存储器

SRAM和DRAM的对比：

|  | SRAM | DRAM |
| :----:| :----: | :----: |
| 存储信息 | 触发器，双稳态电路 | 电容，栅极电容 |
| 破坏性读出 | 非 | 是 |
| 读出后是否需要重写 | 不需要 | 需要 |
| 运行速度 | 快 | 慢 |
| 集成度 | 低 | 高 |
| 存储成本 | 高 | 低 |
| 集成度 | 不需要 | 需要 |
| 是否刷新 | 不需要 | 需要 |
| 送行列地址 | 同时送 | 分两次送 |
| 主要用途 | Cache | 主存 |

## 虚拟存储器

快表和慢表对比：

|  | 快表 | 慢表 |
| :----:| :----: | :----: |
| 存储位置 | 高速缓冲器 | 内存 |
| 速度 | 快 | 慢 |
| 容量 | 小 | 大 |
| 内容 | 慢表的副本 | 完整 |
